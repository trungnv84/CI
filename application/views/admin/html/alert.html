JAlert

jQuery Plugin, Simple, flexible and powerful!

Simple alert

Css and JS in head tag:
<link href="alert.css" rel="stylesheet" type="text/css" />
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="alert.js"></script>

Html:
<button id="sa1">Example</button>

JavaScript:
<script type="text/javascript" language="javascript">
    $(document).ready(function () {
        $("#sa1").click(function(){
            $.alert("", "Message", "\"<b>Example</b>\" button is clicked", "message", 0);
        });
    });
</script>

Alert one time

Used in case the function is called multiple times, but only display one popup.

<link href="alert.css" rel="stylesheet" type="text/css" />
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>

Html:
<button id="sa3">Example</button>
<button id="sa4">Example 2</button>

JavaScript:
<script type="text/javascript" language="javascript">
    $(document).ready(function () {
        $("#sa3").click(function(){
            $.alert("only", "Message", "\"<b>Example</b>\" button is clicked", "message", 0);
            $.alert("only", "Message", "\"<b>Example</b>\" button is clicked, again", "message", 0);
        });
        $("#sa4").click(function(){
            $.alert("anything", "Message", "\"<b>Example 2</b>\" button is clicked", "message", 0);
            $.alert("anything", "Message", "\"<b>Example 2</b>\" button is clicked, again", "message", 0);
        });
    });
</script>

Confirm popup

Popup with the button and the button click event. 

Css and JS in head tag:
<link href="alert.css" rel="stylesheet" type="text/css" />
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="alert.js"></script>

Html:
<div id="msw2"></div>
<button id="sa16">Example</button>
<button id="sa17">Example</button>
<button id="sa18">Example</button>

JavaScript:
<script type="text/javascript" language="javascript">
    $(document).ready(function () {
        $("#sa16").click(function(){
            $.alert("", "Confirm", "\"<b>Example</b>\" button is clicked", "message", 0, "", [
                {
                    label: "Ok",
                    click: function(){
                        this.alertEnd();
                        alert("Accept");
                    }
                },
                {
                    label: "Cancel",
                    click: function(){
                        this.alertEnd();
                    }
                }
            ]);
        });
        $("#sa17").click(function(){
            //Only one time.
            $.alert("anything", "Confirm", "\"<b>Example</b>\" button is clicked", "warning", 0, "#msw2", [
                {
                    label: "Ok",
                    click: function(){
                        this.alertEnd();
                        alert("Accept");
                    }
                },
                {
                    label: "Cancel",
                    click: function(){
                        this.alertEnd();
                    }
                }
            ]);
        });
        $("#sa18").click(function(){
            //Automatically hidden after 3.5 seconds.
            $.alert("", "", "\"<b>Example</b>\" button is clicked", "success", 3500, "", [
                {
                    label: "Ok",
                    click: function(){
                        this.alertEnd();
                        alert("Accept");
                    }
                },
                {
                    label: "Cancel",
                    click: function(){
                        this.alertEnd();
                    }
                }
            ]);
        });
    });
</script>

Events

Two events: show and hide. 

Css and JS in head tag:
<link href="alert.css" rel="stylesheet" type="text/css" />
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="alert.js"></script>

Html:
<button id="sa19">Example</button>

JavaScript:
<script type="text/javascript" language="javascript">
    $(document).ready(function () {
        $("#sa19").click(function(){
            $.alert("", "Message", "\"<b>Example</b>\" button is clicked", "message", 0, "", "", {
                show:function () {
                    alert("Message show.");
                },
                hide:function () {
                    alert(this);
                }
            });
        });
    });
</script>

Button is clicked?

Determine which button is clicked. 

Css and JS in head tag:
<link href="alert.css" rel="stylesheet" type="text/css" />
<script type="text/javascript" src="https://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
<script type="text/javascript" src="alert.js"></script>

Html:
<button id="sa20">Example</button>

JavaScript:
<script type="text/javascript" language="javascript">
    $(document).ready(function () {
        $("#sa20").click(function(){
            $.alert("", "Message", "\"<b>Example</b>\" button is clicked", "message", 0, "", [
                {
                    label: "Yes",
                    click: function() {
                        this.button = "Yes";
                        this.alertEnd();
                    }
                },
                {
                    label: "No",
                    click: function() {
                        this.button = "No";
                        this.alertEnd();
                    }
                },
                {
                    label: "Cancel",
                    click: function() {
                        this.button = "Cancel";
                        this.alertEnd();
                    }
                }
            ], {
                show:function () {
                    this.button = false;
                },
                hide:function () {
                    alert(this.button);
                }
            });
        });
    });
</script>
